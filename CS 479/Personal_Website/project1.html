Knock Blocks

This page is dedicated to Team Voltron's game, Knock Blocks.

Knock Blocks is a physics puzzle game where the players must remove certain blocks in order to complete a level. Currently, there are 3 level packs for the player to enjoy. Each pack has it's own theme, with 3 levels per pack so far. The object of the game is to remove the Grey Blocks by clicking/touching them. This will in turn topple the rest of the blocks on stage. The goal is to keep the Green Blocks on the stage, but knock the Red Blocks completely off. There is a limited amount of clicks the player has to complete each stage. The player wins when all Red Blocks are knocked off stage. Knocking a Green Block off results in a loss.

The game is currently under development. Below you find some screenshots and a link to the Github project, where you can download and build the game yourself.

Knock Blocks is developed using Unity and Microsoft Visual Studio Code.
