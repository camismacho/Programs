C Compiler

This page describes my Compiler project for CS 370.

This compiler is written using Lex and Yacc. This is the largest solo project I have worked on to date. It can take some simple C concepts such as variable declarations, assignments, print statements, loops, if-then-else, and turn it into executable Intel x86 Assembly. This in turn can be run to produce a working C program. It can also recognize syntax, and take in C code written through the terminal or a file, and will output appropriate errors. It also has small debugging features.
