    3.7.1 - convert NFA to DFA

    a) Fig 3.26 (NFA for (aa*)|(bb*))
    Dstates = { A={0,1,3}, B={2}, C={4} }
    Dtran[A,a] = {2}
    Dtran[A,b] = {4}
    Dtran[B,a] = {2} == B
    Dtran[B,b] = {} == error
    Dtran[C,a] = {} == error
    Dtran[C,b] = {4} == C
    done


    b) Fig 3.29
    Dstates = { A={0}, B={0,1}, C={0,1,2}, D={0,1,2,3} }
    Dtran[A,a] = {0,1} = B
    Dtran[A,b] = {0} = A
    Dtran[B,a] = {0,1,2} = C
    Dtran[B,b] = {0,1} = B
    Dtran[C,a] = {0,1,2} = C
    Dtran[C,b] = {0,1,2,3} = D
    Dtran[D,a] = C
    Dtran[D,b] = D
    done

    c) Fig 3.30
    Dstates = { A={0,1,2,3} }
    Dtran[A,a] = A
    Dtran[A,b] = A

    3.7.3 - convert RegExp to DFA, 2-step process

    a) (a|b)*
    NFA by Thompson's construction has 8 states (1-8)
    - epsilon transitions are: 1->2, 2->3, 2->4, 5->7, 6->7, 7->2, 7->8
    - 'a' transition is 3->5, 'b' transition is 4->6
    - start is 1 and final state is 8
    - [note, you may have numbered differently]
    Convert to DFA:
    Dstates = { A={1,2,3,4}, B={5,7,8,2,3,4}, C={6,7,8,2,3,4} }
    Dtran[A,a] = {5,7,8,2,3,4}=B
    Dtran[A,b] = {6,7,8,2,3,4}=C
    Dtran[B,a] = B
    Dtran[B,b] = C
    Dtran[C,a] = B
    Dtran[C,b] = C
    done: B and C are both final states since they contain 8

    b) (a*|b*)*
    - have fun!

    c) ((e|a)b*)*
    - have fun!

    d) (a|b)*abb(a|b)*
    - have fun!
